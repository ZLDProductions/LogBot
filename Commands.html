LogBot Commands

Command Documentation Key:

Note: Do not use the brackets in your command.

Symbol(s) Explanation
{param} Required Parameter. Always use these.
[param] Optional parameter. Only use these if you want to.
{param: [choice/choice/...]} Required parameter with given choices. Always use one of the choices given.
[param: [choice/choice/,,,]] Optional parameter with given choices. Use one of them if you want, otherwise: don't.

Often, the commands will use separators. Most of the time, it's a space. Sometimes (in the case of $purge, $translate, and others) the commands will use separators such as the vertical bar (|) or double-ampersand (&&). Pay great attention to what is required.
